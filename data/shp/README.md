This folder contains GIS shapefiles used in the project.
Considering copyright restrictions and their large file sizes, the shapefiles are not included directly in this repository.

Please obtain the required shapefiles from the following links and unzip them into this directory:
- [NextGen_OD_Zone_ESRI_11152022.zip](https://nhts.ornl.gov/od/assets/data/NextGen_OD_Zone_ESRI_11152022.zip)
- [tl_2018_us_county.zip](https://www2.census.gov/geo/tiger/TIGER2018/COUNTY/tl_2018_us_county.zip)

In the unlikely event that these links become unavailable, please contact me to obtain the files.
